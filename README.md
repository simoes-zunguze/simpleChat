
# Simple Chat

## 1. Intializing

The is a monorepo of a simple chat
To start folow the instruction bellow

```bash
  yarn install
  cd client
  yarn install
  yarn build
  cd ..
  yarn dev
```

Go to your browser and access this link:
http://localhost:3001/

## 2. Using with Docker

```bash
  yarn install
  cd client
  yarn install
  yarn  build
  cd ..
  docker-compose up
```

Go to your browser and access this link:
http://localhost:3001/
